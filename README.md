# TicoAutos Backend

Backend API for the TicoAutos project.  
This service provides user authentication using JWT and connects to MongoDB.

## Technologies

- Node.js
- Express
- MongoDB (Mongoose)
- JWT Authentication
- bcryptjs
- dotenv
